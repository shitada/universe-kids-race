# Feature Specification: 太陽系全惑星ステージ拡張・ブースト演出強化・タイトルBGM修正

**Feature Branch**: `005-solar-stages-boost-fx`  
**Created**: 2026-03-15  
**Status**: Draft  
**Input**: User description: "ゲーム体験の改善第4弾。ステージを太陽系の全惑星＋太陽を網羅し最終ステージを地球に変更。タイトル画面BGM修正。ブースト効果音改善。ブースト中の炎エフェクト追加。クールダウン完了通知音追加。"

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 太陽系全惑星を旅するステージ構成 (Priority: P1)

プレイヤーが月からスタートし、水星・金星・火星・木星・土星・天王星・海王星・冥王星・太陽を経て、最終ステージで故郷の地球に帰還する。全11ステージで太陽系のすべての惑星を巡る壮大な冒険を体験できる。各惑星は見た目に特徴があり、到着時にその惑星らしさが視覚的にわかる。

**Why this priority**: ゲームのコアコンテンツであるステージ構成の拡張。水星と金星が抜けていたことで太陽系の旅として不完全だった。地球を最終ステージにすることで「冒険から帰還する」という物語性が生まれる。

**Independent Test**: ステージ1から11まで順にプレイし、各ステージで正しい惑星名が表示され、惑星の3Dモデルが特徴的な外観で描画されることを確認する。

**Acceptance Scenarios**:

1. **Given** ゲームを新規開始する, **When** ステージ1をクリアして次のステージに進む, **Then** ステージ2は「水星をめざせ！」と表示される
2. **Given** ステージ2（水星）をクリアする, **When** 次のステージに進む, **Then** ステージ3は「金星をめざせ！」と表示される
3. **Given** ステージ10（太陽）をクリアする, **When** 次のステージに進む, **Then** ステージ11は「地球をめざせ！」と表示され最終ステージとなる
4. **Given** ステージ11（地球）をクリアする, **When** エンディングに遷移する, **Then** ゲームクリアの演出が表示される
5. **Given** 各惑星のステージに到達する, **When** 惑星の3Dモデルが画面に描画される, **Then** 惑星ごとに特徴的な外観（地球は青い海・白い雲・茶色い大陸、木星はオレンジ～茶色の縞模様、土星はリング付き、天王星は水色で横倒しリング、海王星は濃い青、太陽は金色に光りパルスする、水星は小さく灰色でクレーター感、金星は黄色がかったオレンジ）で表示される

---

### User Story 2 - タイトル画面でBGMが流れる (Priority: P1)

プレイヤーがゲームを起動してタイトル画面が表示された後、画面のどこかをタッチすると音楽が流れ始める。現在はBGMが流れないため、ゲームの雰囲気づくりができていなかった。

**Why this priority**: ゲームの第一印象を決めるタイトル画面で音楽が流れないのは致命的な体験欠損。ブラウザの自動再生制限への対応としてユーザーの最初のタッチ操作でオーディオを初期化する。

**Independent Test**: ゲームを起動しタイトル画面を表示。画面をタッチするとタイトルBGMが再生されることを確認する。

**Acceptance Scenarios**:

1. **Given** ゲームを起動しタイトル画面が表示されている, **When** 画面のどこかをタッチする, **Then** AudioContextが初期化されタイトルBGMの再生が開始される
2. **Given** タイトルBGMが再生中, **When** 「あそぶ」ボタンをタッチしてゲームを開始する, **Then** タイトルBGMが停止しステージBGMに切り替わる
3. **Given** タイトル画面でBGMが再生されている, **When** もう一度タイトル画面に戻る, **Then** タイトルBGMが再び再生される

---

### User Story 3 - ブースト中に迫力ある噴射音が鳴り続ける (Priority: P2)

プレイヤーがブーストを発動すると、「ゴーーー！」という持続する噴射音が鳴り始める。ブーストが有効な間ずっと音が続き、ブースト終了時にフェードアウトして止まる。現在の短いスイープ音では迫力と持続感がなかった。

**Why this priority**: ブーストはゲームの重要なアクション。持続する噴射音によりブースト中の爽快感と緊張感が大幅に向上する。

**Independent Test**: ゲームプレイ中にブーストを発動し、ゴーッという噴射音が3秒間持続し、ブースト終了時にフェードアウトすることを確認する。

**Acceptance Scenarios**:

1. **Given** ブーストが使用可能な状態, **When** ブーストを発動する, **Then** ホワイトノイズベースの噴射音（ローパスフィルタ適用）が再生を開始する
2. **Given** ブーストが発動中, **When** ブースト持続時間中, **Then** 噴射音が途切れることなく鳴り続ける
3. **Given** ブーストが発動中, **When** ブースト持続時間が終了する, **Then** 噴射音がフェードアウトして停止する

---

### User Story 4 - ブースト中にロケットが炎に包まれる (Priority: P2)

プレイヤーがブーストを発動すると、ロケットの周囲にオレンジ〜赤色の炎のようなパーティクルが常時放出され、揺らめく。ブースト終了時にパーティクルが消える。ブースト中であることが視覚的にも明確にわかるようになる。

**Why this priority**: ブーストの視覚的フィードバック強化。噴射音と合わせてブースト体験をリッチにする。

**Independent Test**: ブーストを発動し、ロケット周囲にオレンジ〜赤色のパーティクルが揺らめきながら表示され、ブースト終了時に消えることを確認する。

**Acceptance Scenarios**:

1. **Given** ブーストが使用可能な状態, **When** ブーストを発動する, **Then** ロケット周囲にオレンジ〜赤色のパーティクルが出現し始める
2. **Given** ブーストが発動中, **When** ロケットが移動する, **Then** パーティクルがロケットの位置に追従し炎のように揺らめく
3. **Given** ブーストが発動中, **When** ブースト持続時間が終了する, **Then** パーティクルの放出が停止し既存パーティクルがフェードアウトして消滅する

---

### User Story 5 - ブーストクールダウン完了時に通知音が鳴る (Priority: P3)

ブースト使用後のクールダウンが完了し再びブーストが使用可能になった瞬間、「ピコーン！」という短い通知音が鳴る。プレイヤーはゲーム画面を注視していなくても、音でブーストの再使用可能タイミングを把握できる。

**Why this priority**: ゲームプレイの利便性向上。視覚インジケーターだけでなく聴覚フィードバックを加えることで操作タイミングが掴みやすくなる。

**Independent Test**: ブーストを使用し、クールダウン完了時に短い通知音が鳴ることを確認する。

**Acceptance Scenarios**:

1. **Given** ブースト使用後クールダウン中, **When** クールダウンゲージが満タンになる, **Then** 短い「ピコーン！」通知音が再生される
2. **Given** クールダウン完了直後, **When** 通知音が再生される, **Then** 音は短時間（1秒未満）で自然に終了する

---

### Edge Cases

- ステージ11（地球）クリア後に次ステージへの遷移要求が来た場合、エンディングに正しく遷移する
- タイトル画面でタッチ→即座に「あそぶ」ボタンが押された場合、AudioContext初期化とBGM再生の競合が起きない
- ブースト発動と同時に隕石に衝突した場合、噴射音と炎パーティクルが適切にクリーンアップされる
- ブースト中にステージクリアした場合、噴射音と炎パーティクルが停止する
- 音声がミュートの端末でもゲームプレイに支障がない（音声なしでも全機能動作する）
- セーブデータの互換性：既存のステージ1〜8の進捗データが新ステージ構成に適切にマッピングされる

## Requirements *(mandatory)*

### Functional Requirements

#### ステージ構成

- **FR-001**: ゲームは全11ステージで構成される（月→水星→金星→火星→木星→土星→天王星→海王星→冥王星→太陽→地球）
- **FR-002**: ステージ2に水星、ステージ3に金星が新規追加される
- **FR-003**: ステージ11（地球）が最終ステージとなり、クリア時にエンディングへ遷移する
- **FR-004**: 各ステージの難易度（隕石出現間隔・ステージ長など）はステージ番号の進行に応じて段階的に上がる
- **FR-005**: 新規ステージ（水星・金星・地球）にはそれぞれ固有のBGMが割り当てられる

#### 惑星3Dモデルの特徴付け

- **FR-006**: 地球の3Dモデルは青い海・白い雲・茶色の大陸が視認できる外観にする
- **FR-007**: 木星の3Dモデルはオレンジ〜茶色の縞模様を描画する
- **FR-008**: 土星の3Dモデルはリング付きで描画する（既存機能の維持）
- **FR-009**: 天王星の3Dモデルは水色で横倒しのリングを持つ
- **FR-010**: 海王星の3Dモデルは濃い青色で描画する
- **FR-011**: 太陽の3Dモデルは金色に光り、光のパルスアニメーションを持つ
- **FR-012**: 水星の3Dモデルは小さく灰色で、クレーターのあるテクスチャ感を持つ
- **FR-013**: 金星の3Dモデルは黄色がかったオレンジ色で描画する

#### タイトルBGM

- **FR-014**: タイトル画面でユーザーが最初にタッチした時にAudioContextを初期化する
- **FR-015**: AudioContext初期化直後にタイトルBGMの再生を開始する
- **FR-016**: 「あそぶ」ボタン押下時にも従来通りAudioContextの初期化を行うが、すでに初期化済みの場合は再初期化しない

#### ブースト噴射音

- **FR-017**: ブースト発動時に持続する噴射音の再生を開始する
- **FR-018**: 噴射音はホワイトノイズにローパスフィルタを適用して生成する
- **FR-019**: ブースト持続中は噴射音が途切れなく再生され続ける
- **FR-020**: ブースト終了時に噴射音がフェードアウトして停止する

#### ブースト炎パーティクル

- **FR-021**: ブースト発動時にロケット周囲にオレンジ〜赤色のパーティクルの放出を開始する
- **FR-022**: パーティクルはロケットの位置に追従し、炎のように揺らめく動きをする
- **FR-023**: ブースト終了時にパーティクルの新規放出を停止し、既存パーティクルをフェードアウトさせる

#### クールダウン完了通知音

- **FR-024**: ブースト使用後のクールダウンが完了した瞬間に短い通知音を再生する
- **FR-025**: 通知音は「ピコーン！」のような明るく短い音で、1秒未満で終了する

### Key Entities

- **ステージ（Stage）**: ゲーム進行の各段階。番号（1〜11）、目的地惑星名、表示名、絵文字、惑星色、ステージ長、隕石出現間隔、星密度を持つ
- **惑星3Dモデル（Planet Model）**: 各ステージの目的地を表す3Dオブジェクト。惑星ごとに固有の色、形状特徴（リング・縞模様・クレーター等）、アニメーション（パルスなど）を持つ
- **ブーストエフェクト（Boost Effect）**: ブースト発動中の視聴覚演出。噴射音（持続音）と炎パーティクル（視覚）の2つで構成される
- **通知音（Notification Sound）**: ゲーム内イベント発生時に再生される短い音。クールダウン完了時に使用

## Assumptions

- 水星と金星のBGMは既存のBGMパターン（4レイヤー構成）と同じ仕組みで新規作成する
- 水星のステージ長と難易度は月と火星の中間（ステージ長1100、隕石間隔2.8秒程度）
- 金星のステージ長と難易度は水星と火星の中間（ステージ長1150、隕石間隔2.6秒程度）
- 地球のステージは最終ステージとして最高難易度（ステージ長2700、隕石間隔0.5秒程度）
- 既存のセーブデータ（ステージ進捗）は新ステージ構成に合わせてマイグレーションが必要になる可能性がある。旧ステージ2（火星）以降のクリア状況を新ステージ番号にマッピングする
- 噴射音のフェードアウト時間は0.3〜0.5秒程度
- 炎パーティクルは既存のParticleBurstシステムを拡張して実装される想定
- ブラウザの自動再生ポリシーに対応するため、AudioContext初期化はユーザーのタッチイベントの中で行う必要がある

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 全11ステージを初回プレイで順にクリアでき、ステージ11（地球）クリアでエンディングに到達する
- **SC-002**: 各惑星の3Dモデルが一目で何の惑星かわかるほど特徴的に描画される
- **SC-003**: タイトル画面で最初のタッチ後1秒以内にBGM再生が開始される
- **SC-004**: ブースト発動中は噴射音が途切れなく再生され、終了後0.5秒以内にフェードアウトが完了する
- **SC-005**: ブースト中のロケット周囲に炎パーティクルが常時表示され、ブースト終了後1秒以内に全パーティクルが消滅する
- **SC-006**: クールダウン完了時に通知音が再生され、プレイヤーが画面を見ずとも再使用可能タイミングを把握できる
- **SC-007**: すべての音声演出はミュート端末でもゲームプレイに影響を与えない（音声なしでも全操作が可能）
