## User Scenarios & Testing *(mandatory)*

### User Story 1 - 🪐 わくせいずかん（惑星図鑑） (Priority: P1)

ステージをクリアするたびにその惑星の「ずかんカード」がもらえる。カードにはひらがなで書かれた豆知識が載っていて、子どもが読んで楽しめる。タイトル画面に「ずかん」ボタンが追加され、集めたカードをギャラリー形式で閲覧できる。まだクリアしていない惑星は「？？？」でロック表示される。全11惑星を集めるコンプリート欲が周回プレイの動機になり、天文学の基礎知識と読む力を育てる。

**Why this priority**: 惑星図鑑はゲーム全体のリプレイバリューを大幅に向上させる中核的な収集要素。ステージクリアの報酬に教育的価値を追加し、ゲームの目標を「全ステージクリア」から「全カード収集」に拡張する。エイリアンコンパニオンとは独立して動作し、単体でも十分な価値を提供する。

**Independent Test**: ステージ1をクリアして月のずかんカードを獲得し、タイトル画面の「ずかん」ボタンから図鑑を開いて、月のカードが豆知識とともに表示されること、未クリアのステージがロック表示されることを確認する。

**Acceptance Scenarios**:

1. **Given** プレイヤーがステージ1（月）をクリアする, **When** ステージクリア演出が完了する, **Then** 月の「ずかんカード」が獲得され、獲得演出が表示される
2. **Given** タイトル画面が表示されている, **When** 「ずかん」ボタンをタップする, **Then** 惑星図鑑画面が表示され、11惑星分のカード枠が一覧される
3. **Given** 惑星図鑑画面が表示されている, **When** クリア済みの惑星カードをタップする, **Then** カードが拡大表示され、惑星名・絵文字・ひらがなの豆知識が読める
4. **Given** 惑星図鑑画面が表示されている, **When** 未クリアの惑星カード枠を見る, **Then** 惑星名が「？？？」と表示され、カード内容がロックされている
5. **Given** すでに月のカードを持っている状態, **When** ステージ1を再度クリアする, **Then** カードの重複獲得は発生せず、既存のカードが保持される
6. **Given** 全11ステージをクリア済み, **When** 惑星図鑑画面を開く, **Then** 全11枚のカードがアンロック状態で表示され、コンプリート状態が分かる
7. **Given** 惑星図鑑画面が表示されている, **When** 「もどる」ボタンをタップする, **Then** タイトル画面に戻る

### User Story 2 - 🛸 うちゅうのなかま（エイリアンコンパニオン） (Priority: P1)

各惑星のステージをクリアすると、その惑星をテーマにしたかわいいエイリアンの仲間が1体加わる（全11体）。以降のステージでは仲間が宇宙船の周りを飛ぶ（オービット動作）。仲間が増えるたびに星の引き寄せ範囲が少し広がる（base + 仲間数 × 0.2）という小さな特典がある。最終ステージ（地球）では11体の仲間に囲まれて壮観な見た目になる。各仲間は惑星テーマカラーで色分けされ、シンプルな幾何学体（球+コーン等）で表現される。達成感・仲間意識を育て、数の増加を視覚的に体験できる。

**Why this priority**: エイリアンコンパニオンはステージクリアに実質的なゲームプレイ報酬（星の引き寄せ範囲拡大）を与え、仲間が増える視覚的な達成感を提供する。周回プレイの動機として図鑑と相乗効果があり、子どもの達成意欲を刺激する中核機能。

**Independent Test**: ステージ1をクリアして月テーマのエイリアンを獲得し、ステージ2で宇宙船の周りにエイリアンが周回飛行していること、星の引き寄せ範囲が拡大していることを確認する。

**Acceptance Scenarios**:

1. **Given** プレイヤーがステージ1（月）をクリアする, **When** ステージクリア演出が完了する, **Then** 月テーマのエイリアンコンパニオンが仲間に加わり、獲得演出が表示される
2. **Given** 月のコンパニオンを1体持っている, **When** ステージ2を開始する, **Then** 宇宙船の周りを月テーマのエイリアンが周回飛行（オービット）している
3. **Given** コンパニオンが3体いる状態, **When** ステージ4を開始する, **Then** 3体のコンパニオンが宇宙船の周りをそれぞれ異なる軌道で周回飛行している
4. **Given** コンパニオンが1体いる状態（仲間数=1）, **When** 星の引き寄せ範囲を確認する, **Then** 引き寄せ範囲が base + 0.2 に拡大している
5. **Given** コンパニオンが5体いる状態（仲間数=5）, **When** 星の引き寄せ範囲を確認する, **Then** 引き寄せ範囲が base + 1.0 に拡大している
6. **Given** すでに月のコンパニオンを持っている, **When** ステージ1を再度クリアする, **Then** コンパニオンの重複獲得は発生せず、既存の仲間が保持される
7. **Given** 全11ステージをクリア済み, **When** 最終ステージ（地球）をプレイ中に観察する, **Then** 11体のコンパニオンが宇宙船の周りを飛んでおり壮観な見た目になっている
8. **Given** コンパニオンが周回飛行中, **When** 宇宙船が左右に移動する, **Then** コンパニオンが宇宙船に追従してオービットの中心が移動する
9. **Given** コンパニオンが周回飛行中, **When** 隕石が接近する, **Then** コンパニオンは見た目のみの存在であり、隕石との当たり判定は発生しない

### Edge Cases

- 図鑑データとコンパニオンデータの保存がセーブデータの上限（localStorage 5MB）を超えないこと
- セーブデータが破損した場合、図鑑・コンパニオンデータはデフォルト（全未獲得）にリセットされる
- ステージクリア直後にブラウザがクラッシュした場合を考慮し、カード獲得・コンパニオン獲得はステージクリア確定時に即座に保存される
- 図鑑画面でのカードタップ時、複数カードの同時タップは最初のタップのみ受け付ける
- コンパニオン11体が同時にオービットしているときのパフォーマンスがiPad Safariで60fpsを維持できること
- コンパニオンのオービット軌道が宇宙船やお互いと重なりすぎないよう適切に分散される
- 新規プレイヤー（クリア済みステージなし）がタイトル画面で「ずかん」を開くと、全カードがロック状態で表示される
- ステージ途中でブラウザを閉じた場合、そのステージのカード・コンパニオンは獲得されない（クリア必須）
- 星の引き寄せ範囲拡大がゲームバランスを大きく崩さないこと（最大 base + 2.2 で11体全員分）

## Requirements *(mandatory)*

### Functional Requirements

#### わくせいずかん（惑星図鑑）

- **FR-001**: タイトル画面に「ずかん」ボタンが表示され、タップすると惑星図鑑画面に遷移する
- **FR-002**: 惑星図鑑画面に全11惑星分のカード枠がギャラリー形式で表示される
- **FR-003**: ステージクリア時に該当惑星の「ずかんカード」が獲得され、獲得状態がセーブデータに保存される
- **FR-004**: クリア済み惑星のカードには惑星名・絵文字・ひらがなの豆知識が表示される
- **FR-005**: 未クリア惑星のカードは惑星名が「？？？」と表示され、内容がロック状態で表示される
- **FR-006**: クリア済み惑星のカードをタップすると拡大表示され、豆知識の全文が読める
- **FR-007**: 惑星図鑑画面に「もどる」ボタンがあり、タップするとタイトル画面に戻る
- **FR-008**: 各惑星の豆知識はすべてひらがなで記載される（対象年齢5〜10歳）
- **FR-009**: 同じステージを再クリアしてもカードの重複獲得は発生しない
- **FR-010**: ステージクリア時のカード獲得は即座にセーブデータに永続化される

#### うちゅうのなかま（エイリアンコンパニオン）

- **FR-011**: ステージクリア時にその惑星テーマのエイリアンコンパニオンが仲間に加わり、獲得状態がセーブデータに保存される
- **FR-012**: 以降のステージで獲得済みコンパニオンが宇宙船の周りをオービット（周回飛行）する
- **FR-013**: 各コンパニオンは対応する惑星のテーマカラーで色分けされる
- **FR-014**: コンパニオンはシンプルな幾何学体（球+コーン等の組み合わせ）で表現される
- **FR-015**: コンパニオンの数が増えるたびに星の引き寄せ範囲が拡大する（base + コンパニオン数 × 0.2）
- **FR-016**: 宇宙船が移動するとコンパニオンのオービット中心が追従する
- **FR-017**: 複数のコンパニオンが同時にオービットする際、各コンパニオンは異なる軌道に分散配置される
- **FR-018**: コンパニオンは見た目のみの存在であり、隕石やその他のオブジェクトとの当たり判定は発生しない
- **FR-019**: 同じステージを再クリアしてもコンパニオンの重複獲得は発生しない
- **FR-020**: コンパニオン獲得はステージクリア確定時に即座にセーブデータに永続化される

#### 共通

- **FR-021**: 既存のセーブデータ（clearedStage）との後方互換性を保ち、図鑑・コンパニオンの獲得状態を追加で保存する
- **FR-022**: セーブデータが破損または不正な場合、図鑑・コンパニオンデータはデフォルト（全未獲得）にリセットされる

### Key Entities

- **ずかんカード（Encyclopedia Card）**: 各惑星に対応する収集カード。惑星名・絵文字・ひらがなの豆知識・獲得状態（ロック/アンロック）を持つ。全11枚
- **コンパニオン（Companion）**: 各惑星テーマのエイリアンキャラクター。惑星テーマカラー・幾何学体の形状・オービット軌道パラメータ・獲得状態を持つ。全11体
- **惑星図鑑画面（Encyclopedia Screen）**: タイトル画面から遷移する閲覧専用画面。全11惑星のカード枠をギャラリー形式で表示し、獲得済みカードの詳細閲覧が可能
- **オービット動作（Orbit Behavior）**: コンパニオンが宇宙船の周りを周回飛行する動作。中心追従・軌道半径・角速度・軌道面の傾きで定義される

## Assumptions

- 惑星図鑑画面はオーバーレイではなく独立した画面（SceneTypeの追加 or 既存UI上の全画面表示）として実装される想定
- コンパニオンの幾何学体は各惑星で異なる組み合わせ（例: 月=球体、土星=球+リング、太陽=球+コーン放射）を想定
- オービット軌道半径はコンパニオン数に応じて自動調整され、11体でも視認性を保つ想定
- 星の引き寄せ範囲の base 値は現在の実装値をそのまま利用し、追加分のみコンパニオン数で決定される
- カード獲得演出は短いアニメーション（1〜2秒程度）で、ゲームフローを阻害しない想定
- コンパニオンの獲得演出もステージクリア演出の一部として短時間で完了する想定
- 図鑑のギャラリーレイアウトはiPad横画面に最適化され、スクロールなしで11枚が一覧できる想定
- セーブデータの拡張は後方互換性を保ち、既存の clearedStage プロパティに影響しない
- コンパニオンの描画負荷を抑えるため、1体あたりのジオメトリは極めてシンプル（10〜50ポリゴン程度）に保つ想定

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: ステージクリア後にずかんカードが獲得され、図鑑画面で閲覧できるまでの操作が3タップ以内で完了する
- **SC-002**: 図鑑画面で全11惑星のカード枠が視認でき、クリア済み/未クリアの状態が一目で区別できる
- **SC-003**: 5〜10歳の子どもが惑星の豆知識をひらがなで読める
- **SC-004**: ステージクリア時にコンパニオンが獲得され、次のステージで宇宙船の周辺を飛んでいることが視覚的に確認できる
- **SC-005**: コンパニオン11体が同時にオービットしている状態でiPad Safariにて60fps以上を維持する
- **SC-006**: コンパニオン数に応じた星の引き寄せ範囲拡大が正しく機能し、プレイヤーが収集の恩恵を体感できる
- **SC-007**: ブラウザを閉じて再度開いても、獲得済みのカード・コンパニオンの状態が保持される
- **SC-008**: 全ステージクリア済みプレイヤーが図鑑の全カードコンプリートとコンパニオン11体勢揃いの達成感を得られる
